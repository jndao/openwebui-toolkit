# OpenWebUI Toolkit

Welcome to my OpenWebUI toolkit!

This repo stores all the artifacts that I have developed to enrich the experience on OpenWebUI. This currently includes

- Functions (filters, actions, pipes)

## Resources

- [OpenWebUI](https://github.com/open-webui)
- [OpenWebUI Repo](https://github.com/open-webui/open-webui)
- [OpenWebUI Functions Docs](https://docs.openwebui.com/features/extensibility/plugin/functions/)

## Development

This code has been written with assistance from AI.

All resources have been developed and is expected to be supported on [OpenWebUI v0.8.10](https://github.com/open-webui/open-webui/releases/tag/v0.8.10) or higher.

No external dependencies (outside ones used by open-webui). See [requirements.txt](https://github.com/open-webui/open-webui/blob/main/backend/requirements.txt)

### Deployment

These functions are used alongside other scripts (which also inspire this repo).

- [OpenWebUI Extensions by @Fu-Jie](https://github.com/Fu-Jie/openwebui-extensions)
- [open-webui-extensions by @Davixk](https://github.com/Davixk/open-webui-extensions)

OpenWebUI is currently deployed on a Proxmox LXC running Debian 12. See [Proxmox Community Scripts](https://community-scripts.org/scripts/openwebui)
